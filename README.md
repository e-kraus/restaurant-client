# Lotus - (client)

> MarketPlace Ecommerce Application

## Quick Installation

```bash
### Clone the git repository
git clone https://github.com/e-kraus/lotus-client.git

### Setup environment variables
cp .env.example .env

### Install project dependencies
npm install

### Initialize development server
npm run serve

### Run production
npm start
```
